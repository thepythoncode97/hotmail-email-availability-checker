## Key Features:
1. **Accurate Hotmail & Outlook Availability Checking**: Checks whether an email address can be newly registered on Microsoft services.
2. **Multiprocessing Support**: Speeds up large email list processing using multiple CPU cores.
3. **HTTP Proxy Support**: Use proxies to avoid rate-limits and increase reliability.
4. **Automatic Result Sorting**  
   - `You_Can_Create_Account.txt`  
   - `You_Can_Not_Create_Account.txt`
5. **High Stability & Error Handling**: Handles request timeouts, proxy failures, and retry logic.
